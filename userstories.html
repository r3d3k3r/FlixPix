User Stories

Guest

  • As a guest, I wish to see the features FlixPicks has to offer.
  • As a guest, I wish to know how FlixPicks works.
  • As a guest, I wish to know whether FlixPicks will be useful for me.

Registered User

  • As a registered user, I wish to be able to have FlixPicks randomly select a movie for me to watch.
  • As a registered user, I want to be able to see all of my subscribed-to streaming services in one place.
  • As a registered user, I wish to be able to leave comments during the movie and see the most popular parts of the movie.
  • As a registered user, I wish to be able to input multiple movie options and have FlixPicks randomly select an option.
  • As a registered user, I wish to be able to see a personalized library of movies based on my interests that I can manipulate.

Streaming Service Representative

  • As a representative of one of the streaming services, I want to be able to see the analytics gathered from FlixPicks.
  • As a representative of a streaming service, I want to see how FlixPicks has recommended my content and the data surrounding user viewing.
  • As a representative of a streaming service, I want to know whether FlixPicks has influenced anyone to subscribe to my service.

Admin

  • As an admin, I need to be able to collect data on user habits (favorite genre, favorite streaming service, even how often each FlixPicks feature is used).
  • As an admin, I would like to see the same types of data that the streaming service representatives would see.
